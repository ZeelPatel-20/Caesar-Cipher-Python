# Caesar Cipher Encryption & Decryption Tool

## Overview

The Caesar Cipher Encryption & Decryption Tool is a Python-based application that demonstrates the implementation of the classical Caesar Cipher algorithm. This project enables users to securely encrypt and decrypt text by applying a configurable shift value to alphabetic characters while preserving spaces, numbers, and special characters.

The project was developed to strengthen fundamental programming skills and provide practical exposure to basic cryptographic concepts used in cybersecurity.

---

## Features

- Encrypt plaintext using the Caesar Cipher algorithm
- Decrypt ciphertext using the same shift value
- Supports both uppercase and lowercase letters
- Preserves spaces, numbers, and special characters
- Validates user input for the shift value
- Simple and interactive command-line interface

## Installation

Clone the repository using Git:

```bash
git clone https://github.com/ZeelPatel-20/Caesar-Cipher-Python.git
```

Navigate to the project directory:

```bash
cd Caesar-Cipher-Python
```

---

## Usage

Run the application using Python:

```bash
python caesar_cipher.py
```

Follow the on-screen instructions to:

- Enter a message
- Specify the shift value
- Select Encrypt or Decrypt mode
